# UGET INSPIRES FOUNDATION Website

## Overview

UGET Foundation is a responsive charity and community-focused website designed to showcase the foundation's mission, programs, impact, and ways people can support its work.

The website provides visitors with information about the organization, highlights ongoing initiatives, displays impact statistics, shares testimonials, and allows users to contact the foundation or subscribe to updates.

---

## Features

### 🌍 Responsive Design

* Fully responsive layout for desktop, tablet, and mobile devices.
* Mobile-friendly navigation menu.
* Optimized user experience across different screen sizes.

### 🏠 Homepage Sections

* Hero section introducing the foundation.
* About section explaining the organization's mission.
* Programs and services showcase.
* Impact statistics with animated counters.
* Testimonials slider.
* Contact section.
* Newsletter subscription area.

### ✨ Interactive Features

* Loading animation.
* Sticky navigation header.
* Active navigation highlighting while scrolling.
* Scroll reveal animations.
* Animated statistics counters.
* Testimonial carousel.
* Dark mode toggle.
* Back-to-top button.
* Client-side form validation.

### 📩 Forms

The website includes:

* Contact form validation.
* Newsletter subscription validation.
* Required field checking.
* Email format verification.

## Installation and Setup

### 1. Clone or Download the Project

Download the project files to your computer.

### 2. Open the Project

Open the folder using a code editor such as Visual Studio Code.

### 3. Run the Website

You can open `index.html` directly in your browser or use VS Code Live Server:

1. Install the Live Server extension.
2. Right-click `index.html`.
3. Select **Open with Live Server**.
